
### Blender 3D printing settings
* Switch Units to Metric
* Set length to Millimeters
* Set Unit scale to 0.0001
* Set scale of view to 0.1 (makes orthographic grid scale properly to above dimensions )
* Set clip start to 1mm
*  Set clip end to 1000000mm
* Install 3D print toolbox extension 
* Keymap space bar to 'search'
* change snapping to grid
* create this all as a new workspace named 3D printing 
* save startup file as default (file -> defaults -> save startup file)
